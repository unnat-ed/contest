November 2018

Web Concour

A mega front-end competition is being organised by Codificaso in collaboration with Inside, News and Media Club, IIT Bhilai.

Rules

  • Functional and responsive front-end : Problem statement will be provided for which you've to build a functional front-end design, which will be responsive (made for both mobile and desktop view).
  • Evaluation Parameters : Front End evaluation is mostly based on the following parameters - functionality of the code, design aesthetics and the quality of code written.
  • Features : In the UI Problem statement, you need to build all the necessary features that are given in the problem statement. You can also add other features of your own, extra points will be given per features depending upon its usability.
  • Originality : You have to design the UI on your own. If any UI is found to be entirely copied, marks will be deducted.
  • Frameworks : You can use any of the frameworks, such as Bootstrap, Material UI, Semantic UI, Materialize, Foundation and so on, provided on the internet.
  • Extra marks: Extra marks will be given to the participants who make their own custom features from scratch.
